### 1.1 查询图书用例
源码如下：

``` 
@startUml test4-1
title 图书查询
hide footbox
skinparam backgroundColor #ded
actor user
boundary 图书查询窗口
activate user
activate 图书查询窗口
user -> 图书查询窗口: 输入检索条件
图书查询窗口 -> 图书查询窗口: 验证输入信息
user -> 图书查询窗口: 点击查看详情
database book
图书查询窗口 -> book: 查询图书
activate book
book --> 图书查询窗口: 返回图书
@endUml
```
